B2B Система электронного размещения заказов ПРАЙД

Использование средств Интернет-сайта www.pride.ru и шаблонных Excel-файлов для оперативного размещения заказов и обозначения потребностей клиентов.

Цели B2B Системы

  • Повышение качества обслуживания клиентов
  • Сокращение времени обработки заказа
  • Предоставление партнёрам возможности самостоятельно размещать заказы в учётной системе Pride (удалённо, через сеть Интернет) и контролировать ход их выполнения
  • Предоставление партнёрам доступа к истории размещённых через Систему заказов

Доступ в B2B Систему

Доступ осуществляется непосредственно через Интернет-сайт ТФГ «Прайд» www.pride.ru (раздел «Сделать заказ» главного меню)

Начало работы с заказами

  1. Выберите необходимый товар из каталога или загрузите спецификацию заказа из excel-файла
  2. Перейдите в раздел "Текущий заказ" и нажмите "Оформить"
  3. Наш менеджер свяжется с Вами по телефону или e-mail, которые Вы указали в регистрационных данных

Выбор товаров по каталогу

Предоставляется возможность выбора товара по логически и удобно сгруппированным разделам и подразделам каталога.

Раздел каталога с товарами

Из данного раздела предоставляется доступ к "карточкам" товара с подробным описанием и возможностью непосредственного добавления в "Текущий заказ".

Карточка товара

Подробное описание товара и возможность удобного добавления нужного количества в заказ

Текущий заказ

  • Удобные возможности:
  • Поиск по любым вхождениям символов в коде товара и наименовании
  • Удаление помеченных позиций
  • Изменение количества в Текущем заказе
  • Переход в «карточку товара» (подробное описание) простым щелчком по наименованию

История заказов

  • Удобные возможности:
  • Доступ ко всей истории заказов, размещённых через Интернет
  • Переход в спецификацию заказа, нажав на его номер
  • Возможность быстро создать повторный аналогичный заказ

Возможности для постоянных клиентов

Возможность проверки наличия необходимых количеств на складе Цены показаны по прайс-листу, привязанному именно к данному клиенту

Поиск товара в каталоге

  • Удобный поиск с возможностью выбора:
  • по коду товара
  • по любым вхождениям символов в наименовании
  • по производителю

Возможность заполнения заказа из файла Excel

Удобная возможность быстро внести в заказ большое количество позиций по предложенному шаблону.
